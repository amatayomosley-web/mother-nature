# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct.

The full text is available at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be civil, focus on the work, treat other contributors with respect regardless of background or experience level. Disagreement about design is welcome and expected; personal attacks and bad-faith engagement are not.

## Project-specific notes

- **The work is the work.** Critique designs, mechanics, and arguments freely. Critique people only via their conduct, never via their identity.
- **Strong disagreement is fine.** Sharp critique is fine. Mockery, dismissiveness, or hostility toward contributors is not.
- **William makes the final calls** on design direction. Disagreement is welcome through discussion; persistence past a final decision crosses into a conduct issue.

## Reporting

If you experience or witness behavior that violates this code, contact William directly through a private channel (GitHub email, or via a contact link if added to the repo). Reports will be handled confidentially.

## Enforcement

Behavior that violates this code may result in: a private warning, removal from a discussion, blocking from the repository, or escalation to GitHub's abuse-reporting channels for serious cases.

The full Contributor Covenant 2.1 includes detailed enforcement guidelines that this project follows.
