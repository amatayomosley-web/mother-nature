# Causal chains

This folder holds notes on specific cause-and-effect chains across systems. Where individual system notes describe what ONE thing does, chain notes describe what happens when TWO OR MORE systems interact across time.

## Why this matters

System interaction graphs are easy to lose track of. "Player over-hunts deer → predator hunger increases → predator aggression rises" spans three system notes. Without explicit chain tracking, this multi-system causality stays invisible until something breaks ("why are bears suddenly aggressive?").

## Conventions

- **One file per distinct causal chain.** If two or more systems are interacting in a specific named pattern, it gets a chain note.
- **Sequence is explicit and ordered.** Each chain lists the steps in order, with the systems involved bracketed via wikilinks `[[system-name]]`.
- **Time scale is named.** Some chains fire instantly (within an encounter); others span days or weeks.
- **Player-readable signals are diegetic only.** No UI alerts. The player learns the chain through observable in-world consequences and their own compendium notes.
- **Counter-actions are listed.** What the player can do to break or redirect the chain.
- **Tags** classify: `#causal-chain`, `#ecology`, `#weather`, `#player-pressure`, `#predation`, etc.
